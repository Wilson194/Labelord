Labelord
=========

This application is simple tool for working with GitHub labels. You can easily create new label, delete label, edit some labels
and what is the best, replicated all labels through repositories!

This project was develop as homework for CTU FIT for lecture MI-PYT.

Whole project is programed in python and using GitHub API. This application works only with GitHub!
